# 小技巧

1. 指令查詢： `help`, `doc`, `lookfor`. (`lookfor` 很慢，建議使用 Google 大神) 
2. 輸入一部份指令後，使用 `Tab` 鍵 MATLAB 可以列出所有可能的指令，如果指令只有一個則會自動補齊。
